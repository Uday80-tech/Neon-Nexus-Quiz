# **App Name**: Neon Nexus Quiz

## Core Features:

- Dynamic Quiz Generation: Fetch quiz questions based on the topic selected by the user from a third party API.
- Interactive 3D Environment: Display an animated 3D environment using Three.js to create an immersive quiz experience.
- Real-time Score Tracking: Calculate and display the user's score in real-time as they progress through the quiz.
- Leaderboard Ranking: Maintain a leaderboard of top scores using Firestore, allowing users to compete and track their performance.
- AI-Powered Difficulty Adjustment: Use an AI tool to analyze user performance and adjust the quiz difficulty dynamically for an optimized learning experience.
- Personalized Learning Paths: Suggest additional quizzes or learning resources based on the user's quiz history and performance.
- User Account Creation: Allow users to create accounts and track their performance over time using Firebase Auth.

## Style Guidelines:

- Primary color: Vibrant cyan (#34D1BF) to evoke a sense of futuristic technology and energy.
- Background color: Dark, desaturated gray (#222A35) to provide contrast and highlight the neon elements.
- Accent color: Electric purple (#A239EA) for interactive elements and highlights, creating visual interest.
- Font for headlines and body: 'Space Grotesk', a sans-serif font, providing a modern, tech-inspired look.
- Code font: 'Source Code Pro' for displaying code snippets, giving a retro, terminal-like feel.
- Use glowing, minimalist icons to represent different quiz topics.
- Employ smooth, elegant transitions for question changes.
